# The Cut v2.0 — Home Screen Preview

This is a standalone Expo project containing ONLY the new v2.0 home screen,
wired to your existing live Railway backend (the same one the current app uses).

No Leaderboard, Creators tab, or My Golf screen — just the Home feed, so you
can preview the new design and navigation feel before committing more build time.

See SETUP_STEPS.md for how to get this running in Expo Go.
